# BUS Ticket Management Project

A comprehensive Bus Ticket Management System built with Django and Django REST Framework, featuring real-time updates, task scheduling, and a PostgreSQL backend.

---

## 🛠 Technical Stack

This project uses the following technologies:

1. **Django** – High-level Python web framework for rapid development  
2. **Django REST Framework** – For building RESTful APIs  
3. **PostgreSQL** – Relational database management system  
4. **Celery** – Asynchronous task queue for background jobs  
5. **Redis** – In-memory data store for caching and message broker  
6. **Docker** – Containerization for development and production  
7. **GitHub Actions** – CI/CD workflow automation

---

## ⚡ Features

- User authentication and role-based access control  
- Real-time ticket booking and availability updates  
- Background task processing with Celery  
- Containerized environment for easy deployment 
- Adding Proper testing using Django default unit test
